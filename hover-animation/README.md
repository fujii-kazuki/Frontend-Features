# ホバーアニメーション
CSSの擬似クラス「:hover」、擬似要素「::before, ::after」、「transition」プロパティを使用したアニメーションを実装し、これらの理解が深まりました。  
また、「perspective」プロパティを使用する事で、アニメーションに奥行きをもたらし、表現力の幅を広げる事もできました。

使用技術：HTML / CSS  
作業時間：2h
